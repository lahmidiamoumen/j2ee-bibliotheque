# j2ee-bibliotheque
Projet J2EE simple site un seul admin qui peut ajouter des livres ou des auteures

dans mon exapmle j'ai remplacer les livres par des produits 

### Architecture used 
-	 MVC with DOA

### Password hash type 2
-	 SHA-256 
### Devlopment Tools:
-	 Intellij IDEA Ultimate 2019.3
### Database:
-	 Mysql
